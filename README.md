# Py-To-Do-List

A simple, object-oriented Python todo list application for managing tasks across multiple lists with JSON persistence.

## Files
- README.md
- Todo.py        # Core application (classes: Task, TodoList, TodoManager)
- todo_data.json # Auto-generated data storage

## Quick Start
1. Clone the repo:
   git clone https://github.com/reezmahanan/py-todo-list.git
   cd py-todo-list

2. Run the app:
   python Todo.py

## Features
- Create multiple todo lists
- Add, remove, and mark tasks complete
- Task priorities (High / Medium / Low)
- Save and load data from todo_data.json

## Data
Tasks and lists are stored in todo_data.json in a simple JSON structure for easy persistence and portability.
